# CSharpProject

Это приложение представляет собой дневник питания, который помогает пользователям отслеживать ежедневное потребление калорий и питательных веществ.

## Архитектура и логика

Приложение построено на базе .NET Windows Forms и использует локальную базу данных SQLite для хранения информации.

### База данных

- **Инициализация**: При первом запуске приложение автоматически создаёт файл `nutrition.db`, если он отсутствует.
- **Импорт данных**: Если база данных пуста, приложение импортирует данные о продуктах из `Food_Nutrition_Dataset.csv`.
- **Таблицы**:
  - `food`: Хранит информацию о продуктах, включая калории, белки, жиры, углеводы и другие питательные вещества.
  - `daily_log`: Записывает, какие продукты и в каком количестве пользователь употребил в определённый день.

### Основной функционал

1.  **Поиск и добавление продуктов**:
    -   Пользователь может искать продукты в базе данных.
    -   Выбранный продукт можно добавить в дневник, указав его вес в граммах.

2.  **Дневник питания**:
    -   Отображает список продуктов, съеденных за день.
    -   Автоматически подсчитывает общую сумму калорий и питательных веществ.

3.  **История**:
    -   Позволяет просматривать и редактировать записи за предыдущие дни.

4.  **Калькулятор калорий**:
    -   Помогает рассчитать базальный метаболизм (BMR) и суточную потребность в калориях (TDEE).
    -   Поддерживает три формулы:
        -   **Mifflin-St Jeor**: Учитывает пол, вес, рост и возраст.
        -   **Revised Harris-Benedict**: Альтернативная формула с теми же параметрами.
        -   **Katch-McArdle**: Основана на мышечной массе (требует ввода процента жира).

## Сборка и запуск

Проект можно открыть и запустить в среде разработки, поддерживающей .NET, например, в Visual Studio. Все необходимые зависимости управляются через NuGet.
